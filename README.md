## 📌 Abstract
This project focuses on detecting fake news using deep learning techniques. 
An LSTM-based model combined with GloVe word embeddings is used to classify news articles with high accuracy.

According to the report, the model achieves accuracy above 98%, making it highly effective for real-world applications.

---

## 🎯 Objectives
- Detect fake and real news automatically  
- Improve classification accuracy using deep learning  
- Use word embeddings for better text understanding  
- Build scalable detection system  

---

## 📊 Dataset
- News articles labeled as TRUE / FAKE  
- Text includes title and content  
- Preprocessed using NLP techniques  

---

## ⚙️ Methodology
1. Data Cleaning (remove noise, stopwords)  
2. Text preprocessing and tokenization  
3. Word embeddings using GloVe  
4. Model building using LSTM  
5. Evaluation using accuracy, confusion matrix, ROC  

---

## 🧠 Model Architecture
- Embedding Layer (GloVe)
- Bidirectional LSTM  
- GlobalMaxPooling  
- Dense layer (Sigmoid)

---

## 📈 Results
- Accuracy: ~98%  
- High precision and recall  
- Strong performance in detecting fake news  

---

## 📉 Key Insights
- LSTM captures sequential patterns effectively  
- Word embeddings improve semantic understanding  
- Preprocessing significantly improves accuracy  

---

## ⚠️ Limitations
- Requires large dataset  
- Computationally intensive  
- Needs updates for evolving fake news patterns  

---

## 💡 Future Improvements
